Abstract

The HPC-FAIR AI Model portion of the ontology provides properties and classes relating to describing different machine learning models, such as their composition and various features within them.

Introduction back to ToC

The AI Model portion of the HPC-FAIR Ontology provides two classes to describe a decision tree (along with its nodes) along with various properties. This portion is minimal and can be used in conjunction with other portions of the ontology.
